📖 Ayuda y Manual
Instrucciones, instalación y sincronización
🔄 Cómo funciona la sincronización
• La app usa Firebase (Google) para sincronizar en tiempo real
• Cualquier cambio en el PC aparece en el celular en segundos y viceversa
• En el celular: abre la app → pulsa 🔄 Sincronizar ahora → listo
• Sin internet: los datos se guardan localmente y se sincronizan solos al reconectarse
• El badge 🟢 En línea confirma que la conexión está activa
🤖 Instalar en Android
Opción recomendada — Chrome (sin instalar nada):
1. Abre la app en Chrome para Android
2. Menú ⋮ (tres puntos) → "Añadir a pantalla de inicio"
3. Confirma → el ícono aparece como app nativa ✓
4. Funciona offline, con sincronización automática ✓

Alternativa — Gonative (APK real, gratis):
1. Ve a gonative.io/create desde el PC
2. Pega la URL de la app → selecciona Android
3. Clic "Build App" → recibes el APK al correo en ~5 min
4. En el celular: activa "Instalar desde fuentes desconocidas" → instala
🍎 Instalar en iPhone (iOS)
Paso a paso (Safari obligatorio):
1. Abre la app en Safari (no Chrome, no funciona desde Chrome en iPhone)
2. Toca el ícono de compartir ⬆️ en la barra inferior
3. Desliza y toca "Agregar a pantalla de inicio"
4. Ponle el nombre que quieras → toca "Agregar"
5. El ícono aparece en tu pantalla de inicio ✓

Nota: iOS tiene sincronización automática igual que Android.
Todos los datos se comparten con el PC en tiempo real.
📋 Manual de uso
Flujo básico:
1. Admin/Supervisor: crea un proyecto → entra al módulo → carga la BD (Excel/CSV)
2. Auxiliar: abre el proyecto → selecciona módulo → digita conteo por registro
3. Al guardar el conteo, el status cambia a Contado automáticamente
4. Al exportar Excel, el status cambia a Exportado

Roles:
• 👑 Administrador: acceso total, gestiona usuarios, configuración, exporta
• 🔷 Supervisor: crea proyectos, carga BD, ve stock, exporta
• 👷 Auxiliar: solo digita. Stock oculto (***) para conteo transparente

Fotos:
• Las fotos se marcan automáticamente con Material, PEP, Lote y fecha
• Al guardar, se descargan a una carpeta con el nombre del módulo y la fecha
• Las fotos se incluyen en el Excel al exportar

Cambio de Ubicación:
• Módulo especial con listas desplegables configurables
• Al exportar/enviar correo, el Excel se descarga automáticamente
• El correo se abre con el asunto y cuerpo listos para enviar

IDs automáticos:
• Módulos estándar: INV-00001, INV-00002...
• Cambio de Ubicación: UBK-00001, UBK-00002...
📁 Configurar Google Drive para fotos
Las fotos quedan en Drive de inventarios.vao.gor@gmail.com
Estructura: Inventario_Fotos / NombreProyecto / NombreModulo_Fecha

Pasos (una sola vez):
1. Inicia sesión en script.google.com con inventarios.vao.gor@gmail.com
2. Clic en "Nuevo proyecto" → borra el código existente
3. Pega el código que aparece en los comentarios del archivo .html (busca "function doPost")
4. Clic en Implementar → Nueva implementación → App web
5. "Ejecutar como": Yo · "Quién puede acceder": Cualquier usuario → Implementar
6. Copia la URL que termina en /exec
7. En la app: ⚙️ Configuración → URL Apps Script Drive → pega la URL → Guardar

✅ A partir de ahí, todas las fotos se guardan automáticamente en Drive
☁️ Alojar la app en la nube (para APK)
1. Ve a github.com → New repo → sube index.html + manifest.json + sw.js + icons/
2. Ve a vercel.com → importa el repo → Deploy → obtienes URL pública
3. Con esa URL ve a gonative.io o pwabuilder.com para generar el APK
